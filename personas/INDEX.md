# Persona Catalog

Available AI agent personas for the Benefit & Insurance Navigator.

## Community

| Persona | Purpose |
|---------|---------|
| `community/benefits-researcher` | Discover federal, state, and local benefit programs with source verification |
| `community/insurance-analyst` | Compare insurance across ACA marketplace, off-marketplace, short-term, health sharing, ICHRA/QSEHRA |
| `community/evidence-verifier` | Adversarial fact-checker: independently recalculates FPL, verifies expansion status, cross-checks data consistency |
| `community/eligibility-analyst` | Cross-validate eligibility using verified data, detect coverage gaps, income cliffs, and program interactions |
| `community/action-planner` | Produce prioritized enrollment plans with deadlines, documents, and contingency paths |
